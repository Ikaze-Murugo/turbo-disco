How to Use

1. Include the CSS

Add the simple-styles.css file to your project and include it in your HTML head section.

2. Use the Classes

Apply the CSS classes directly to your HTML elements. No JavaScript or complex components needed.

3. Customize as Needed

Modify the CSS custom properties in the :root section to change colors, spacing, and other design tokens.
